Remark

"01340-Controller.php" contains two scripts: "Controller.Symfony.MySql.php.t4" and "Controller.UserMask.Symfony.MySql.php.t4".

With "Controller.Symfony.MySql.php.t4"-script you generate php CRUD-operations for the given ViewModel.

With "Controller.UserMask.Symfony.MySql.php.t4" you generate two methods:

- getcurrusermasks: Right after login the client app sends the request for security bitmap mask vector for the given user.

- getwithfilter: The Admin-User sends the request for security bitmap mask vector for the selected user.
